进入docker管理mongo数据库

  • 进入docker容器

    docker exec -i -t xxx bash

  • 进入mongo

    mongo
  • 查看数据库
    show dbs
  • 选择数据库
    use Germany
  • 查看表单
    show collections
  • 查看表单内容
    db.germany.find()

  • 删除该表单所有数据

    db.germany.remove( { } )

  • 参考链接：
    http://www.tutorialspoint.com/mongodb/mongodb_create_database.htm
    https://docs.mongodb.org/manual/reference/method/db.collection.remove/
