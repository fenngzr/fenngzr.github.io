一、用户部分接口

1.快速登录

Post

http://domain.com/phone/quicklogin

请求：
参数 参数类型 说明
userInfo String 手机号码或者电子邮箱
password String 密码用户名为手机号则不需要
返回参数
参数 参数类型 说明
code Int 接口执行结果状态 0=成功
userId Long 用户id
userName String 用户名
avatar String 头像
返回结果Json示例

2.注册

Post

http://domain.com/phone/register

请求：
参数 参数类型 说明
email String 电子邮箱
password String 密码
返回参数
参数 参数类型 说明
code Int 接口执行结果状态 0=成功
userId Long 用户id
email String 电子邮箱
返回结果Json示例

3.第三方登陆

Post

http://domain.com/phone/otherLogin

请求
参数 参数类型 说明
userId String 用户id
userName String 用户名
type Int 登录类型
返回参数
参数 参数类型 说明
code Int 接口执行结果状态 0=成功
返回结果Json示例

二、新闻列表页接口

Post

http://domain.com/phone/newslist

请求
参数 参数类型 说明
columnId Long 栏目id：比如推荐栏目为000001
page Long 当前页码，默认为0
size Long 每页的新闻数量
返回参数
参数 参数类型 说明
code Int 接口执行结果状态 0=成功
info String 返回头图信息数据，Json数据
list String 返回普通列表结果数据，Json格式
topList String 返回置顶列表结果数据，Json格式（字段同list）
info返回结果数据字段
参数 参数类型 说明
columnName String 栏目名称
picLinks String 头图链接
describe String 描述是否需要
articleId Long 文章id
artUrl String 详情页链接
list/toplist返回结果数据字段
参数 参数类型 说明
articleId Long 文章id
title String 标题
picLinks String 列表图
pubTime Long 发布时间
sourceName String 文章来源
comments Long 评论数
editor String 文章作者名字
artUrl String 详情页url
images String 文章图片地址（多图以逗号分隔）
newsType Long 新闻类型：0=标题新闻；1=图文新闻；2=视频新闻；3=音频新闻；4=多图；6=单图新闻
返回结果Json示例
