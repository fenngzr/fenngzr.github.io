json文件放在服务器生成url

  • 将相应的文件放到服务器的某个目录下 
    /User/project/ABC.json

  • 配置nginx

    location /whaterver {
        alias /User/project/;

}

    -重启nginx

    sudo service nginx restart

    -打开浏览器，输入以下地址，即可访问

    www.yourdomain.com/whaterver/ABC.json
